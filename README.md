# Cloister

Containment-first local AI coding environment: a jailed coding agent that
builds and tests through a **builder**, writes source only through an audited
**scribe**, and reaches the web only through a quarantined **scholar** — each a
mode of one Go binary, wired into per-project "cells" via Docker Compose.

> **Status:** migrating in from a private Subversion repository, one package
> per PR. See [docs/migration.md](docs/migration.md) for the plan and progress.
> A full README lands at the end of the migration.
